# Estudo-Banco-de-Dados
Repositório dedicado aos meus estudos em Banco de Dados, com foco em SQL Server. Contém exercícios práticos, consultas SQL, criação de tabelas, views, joins, procedures e outros conceitos essenciais para manipulação e análise de dados. O objetivo é consolidar o aprendizado na prática e servir como base para projetos futuros.
